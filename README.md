# Infra
Contains the ansible scripts for all Project

# Repository structure
--------------------

This structure is a default one. You are able to modify it as you wish.

```
<repo-root>/
|
╰- packages/
|  ╰- <app-name>/
|     |- src/
|     | |- ...
|     | ╰- ansible.cfg
|     ╰- README.md
|
╰- vars/
|  |- ....
|  ╰- credentials.anvars/
|
╰- hosts
|
╰- .gitattributes
```
